
# Today's Lunch Menu Recommendation Site

## Overview

This is a simple web application that recommends a random lunch menu to users who have trouble deciding what to eat.

## Features

*   **Menu Recommendation:** Recommends one of a variety of lunch menus.
*   **Random Selection:** The menu is chosen randomly to provide a new experience each time.
*   **Simple UI:** It has a clean and intuitive user interface.

## Project Structure

*   `index.html`: The main page of the application.
*   `main.js`: Contains the logic for the menu recommendation.
*   `style.css`: Defines the styles for the application.
*   `blueprint.md`: This file.
